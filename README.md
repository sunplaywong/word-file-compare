# docx-file-compare
compare two docx file and highlight differences (支持中文各种编码字符）
手头有不少比较软件，不是不支持中文就是运行卡顿，一时兴起用Hermes Agent手搓了一个比较器，欢迎大家使用和多提宝贵意见！Just an AI Generated Tools for DOCX comparing, enjoy!
直接通过Python运行docx_compare.py即可。但需要确保有安装“python-docx”和“regex”模块，安装方式如下：
pip install python-docx
pip install regex
